# ProjetoCRUD1

## Projeto acadêmico CRUD desenvolvido em C#.

## Sistema CRUD desenvolvido em C# com Windows Forms.

#### Funcionalidades
#### Cadastro de usuários
#### Edição de dados
#### Exclusão de registros
#### Consulta de informações
#### Tecnologias utilizadas
#### C#
#### Windows Forms
#### SQL Server

** A realização deste projeto permitiu aplicar na prática os conceitos de CRUD utilizando C#, Windows Forms e banco de dados SQLite. Durante o desenvolvimento, foi possível compreender melhor a manipulação de dados, navegação entre formulários, utilização de eventos e integração com banco de dados.
O sistema desenvolvido atende às funcionalidades propostas, permitindo cadastrar, consultar, atualizar e excluir registros de forma organizada e funcional. Além disso, o projeto contribuiu para o aprimoramento da lógica de programação e da estruturação de aplicações desktop. **
